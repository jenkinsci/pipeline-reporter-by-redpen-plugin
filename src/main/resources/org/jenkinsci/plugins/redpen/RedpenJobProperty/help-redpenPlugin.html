The Redpen Jenkins plugin provides an easy, and secure way to connect and send build and deployment information from your Jenkins server to your Jira sites. The pipeline reports the build and deployment status (failure or success) with other meaningful data directly into the Jira issues which helps your team to track and collaborate.

This plugin gives visibility to your team of the CI/CD pipelines related to Jira issues. The plugin adds the build and deployment related data as Jira issue comments. So, it is important that the branch that triggers the Jenkins pipeline must have the Jira ticket number in the branch name. (e.g. [TEST-1234] Deploy v1.2 sto production)

The plugin in powered by Redpen.
